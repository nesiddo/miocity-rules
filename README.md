# MioCity ルールアーカイブ

FiveM サーバー「**MioCity**」のルールを検索・閲覧できる静的サイトです。

## 公開URL
**https://nesiddo.github.io/miocity-rules/**

## 収録ルール
- **SRV-01** サーバールール（概要・一般・RP・禁止行為）
- **CRM-01** 犯罪ルール（戦闘・車両・PD対応・法律・罰金表・人数表など）
- **GNG-01** ギャングルール（ギャング運営・抗争・島取りルール）
- **DOC-01** 個人医ルール（治療対応・料金・禁止事項）

## 特長
- 全条文のキーワード横断検索（見出し・列名も対象、他ファイルのヒットも案内）
- ライト / ダークのテーマ切替（設定は保存されます）
- スマホ対応（表はカード表示に自動整形）／印刷対応
- サイバーパンク調の UI・アクセシビリティ配慮（コントラスト AA・キーボード操作・reduced-motion）

## 技術
- 依存ライブラリなしの単一 `index.html`
- ルール本文は `<script id="mio-data">` 内の JSON、UI は素の HTML/CSS/JS
- 内容は原文の Google ドキュメントに基づきます（表記は随時変更の可能性あり）
